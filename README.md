Open source ipinfo API.

## Run project locally
Open your terminal.

1. Clone repository
    ```sh
    git clone https://github.com/ReallyFatYoshi/ipinfo.git
    ```

2. Open repository
    ```sh
    cd ipinfo
    ```

3. Install dependancies
    ```sh
    npm install
    ```

4. Run the application
    ```sh
    npm run start
    ```
